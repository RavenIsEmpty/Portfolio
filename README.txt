HOW TO USE (GitHub Pages)

1) Put your photo at: assets/img/profile.jpg
   - square image looks best (e.g., 600x600)

2) Update your links:
   - In index.html, find "GitHub (TODO)" and "Case Study (TODO)" and replace with real URLs.
   - Replace LinkedIn TODO link.
   - Replace CV link (Download CV).

3) Run locally (recommended):
   - VS Code -> install "Live Server" extension
   - Right click index.html -> Open with Live Server

4) Deploy to GitHub Pages:
   - Repo Settings -> Pages
   - Build and deployment: Deploy from a branch
   - Select branch: main (or master) / root
